# 学吧 — 使用指南

## 一句话介绍

学吧是一个 AI 智能学习助手，帮你**从目标到掌握**走完整个学习闭环：规划→学习→测验→调整→复盘。所有内容推送到飞书，数据驱动个性化适配。

---

## 快速上手

### 第 1 步：开启对话

在飞书中找到「学吧」，发送第一条消息。Agent 会主动介绍自己能做什么。

> 💡 不需要寒暄，直接告诉 Agent 你想学什么。

### 第 2 步：描述学习目标

用自然语言描述你的学习目标，例如：

**有明确考试/等级标准的：**
```
我想考中级经济师，零基础，每天能学 1 小时，希望 3 个月内通过
```

**没有明确标准的（技能/语言/兴趣）：**
```
3 个月内流利说英语，能参加英文工作会议
3 个月掌握 LLM 原理，能读懂论文、跑通微调
```

Agent 会先判断你的目标类型：

| 类型 | 特征 | Agent 怎么做 |
|------|------|-------------|
| **有标准型** | 有考纲/证书/等级框架 | 联网获取官方大纲，作为知识树的依据 |
| **无标准型** | 没有外部验收标准 | 了解你的使用场景，帮你定义"怎样算学会"的里程碑 |

对于无标准型目标，Agent 会和你一起定义 3-5 个**由浅到深的里程碑**，每个里程碑都有具体的"能做到什么"作为验收标准。

**项目驱动型**（编程/技术类）：Agent 会主动了解你的工作场景，联网推荐 3-5 个具体项目供你选择，并定义每个项目的交付物标准——不是泛泛的"做个 Web 应用"，而是"做一个命令行记账工具：SQLite 存储，月度统计报表"。

Agent 最终帮你把目标具体化为：

| 要素 | 说明 |
|------|------|
| 学什么 | 具体科目/技能 |
| 到什么水平 | 考试分数 / 里程碑验收标准 |
| 什么时候完成 | 截止日期 |
| 每天多少时间 | 工作日/周末可投入时长 |
| 当前基础 | 零基础/有经验/想深入 |

### 第 3 步：飞书权限授权（仅首次）

首次创建学习项目时，Agent 会发送一个飞书授权链接。**一次性授权全部权限**（完整 scope 清单见项目内 `workspace/templates/feishu-scopes.json`），涵盖：

- 📄 文档读写（学习内容、报表、评论、权限管理）
- 📊 多维表格（学习数据数据库 5 张表的增删改查）
- 📚 知识库（学科目录结构创建和管理）
- 📁 云空间（文件夹创建、文件上传下载、版本管理）
- 💬 消息推送（内容推送、测验通知、交互按钮）
- 👤 通讯录（用户身份识别）
- 📅 日历（学习日程提醒）
- ✅ 任务管理（学习任务追踪）
- 🔍 搜索（文档和消息检索）

> 授权只需一次，后续使用中不再重复请求。

### 第 4 步：确认知识树

Agent 会联网检索该领域的完整知识框架，生成**知识树**（三级结构）供你确认：

```
科目: XXX
|-- 模块 A
|   |-- 知识点 A1
|   |-- 知识点 A2
|   +-- 知识点 A3
|-- 模块 B
|   |-- 知识点 B1
|   +-- 知识点 B2
...

共 X 个模块，Y 个知识点，预计 Z 小时
✅ 验证通过 | 审计通过
```

你可以：
- ✏️ 添加遗漏的知识点
- ❌ 删除不需要的内容
- 🔄 调整优先级
- 🔀 重新分组

### 第 5 步：开始学习

知识树确认后，Agent 自动完成：
1. **生成学习计划**（按艾宾浩斯遗忘曲线排程）
2. **创建飞书目录**（学科文件夹 + 章节子文件夹 + 复习资料/报表）
3. **设置定时任务**（每日推送、复习提醒、每周测验等）
4. **推送第一篇学习内容**到飞书文档

## 日常使用

### 📖 每日学习

Agent 会按计划在飞书推送学习内容卡片：

```
📖 今日学习内容：[知识点名称]

📌 核心要点：
• 要点 1
• 要点 2
• 要点 3

⏱️ 预计学习时间：45 分钟
📄 完整教材：[点击查看飞书文档]

━━━━━━━━━━━━━━━━━━
[✅ 已完成] [⏰ 需要更多时间] [🆘 太难了]
```

点击 **「已完成」** 后，Agent 会记录学习数据、推送随堂小测（1-3 题）、推送下一篇内容。

### 📝 测验

Agent 定期发起测验，通过飞书消息**逐题发送**，你直接回复答案即可。

**测验内容根据目标类型自适应：**
- 考试型 → 选择题/判断题/简答题/应用题，围绕考点和历年真题
- 无标准型 → 场景模拟题/操作题/概念解释题，围绕实际应用场景

```
📝 章节测验：[章节名称]

Q1/5（选择题 · 2分）
━━━━━━━━━━━━━━━━━━
题目内容...

A) 选项 A
B) 选项 B
C) 选项 C
D) 选项 D

请回复 A/B/C/D
```

测验类型：

| 类型 | 触发时机 | 题量 | 目的 |
|------|---------|------|------|
| 随堂测验 | 内容推送后 | 1-3 题 | 检测基本理解 |
| 每日回顾 | 每日定时 | 3-5 题 | 间隔重复防遗忘 |
| 每周回顾 | 每周定时 | 5-10 题 | 综合复习本周内容 |
| 月度评估 | 每月定时 | 10-20 题 | 全面评估掌握度 |
| 考前模拟 | 考前 14 天 | 30-50 题 | 全真模拟 |

###  查看进度

随时问 Agent：

| 你想问 | 说 |
|--------|-----|
| 学习进度 | "我学得怎么样了" / "查看报表" |
| 知识掌握 | "帮我复盘" / "总结一下" |
| 知识树 | "看看我的知识树" |
| 错题 | "看看错题本" |
| 计划 | "我的学习计划是什么" |

Agent 会生成可视化报表：
- **知识图谱看板**：已完成✅ / 学习中🔄 / 未开始
- **掌握度热力图**：颜色深浅表示掌握程度
- **学习时长统计**：日/周/月趋势
- **薄弱点 Top 10**：最需要补强的知识点

### 🔄 动态调整

Agent 根据学习数据**自动调整计划**，每次调整都会告知原因和变化：

| 情况 | 调整 |
|------|------|
| 掌握度 ≥ 80% | 跳过复习，加速推进 |
| 掌握度 60-79% | 按原计划继续 |
| 掌握度 40-59% | 增加专项练习，后续减速 |
| 掌握度 < 40% | 暂停新内容，重新讲解 |
| 连续学得快 | 增加每日推送量 |
| 连续学得慢 | 减少推送量，增加复习 |
| 进度落后计划 | 建议延长截止日期 |

## 飞书集成

### 知识库结构

首次创建学习项目后，Agent 会在飞书自动创建完整的目录结构：

```
📚 学习空间（根知识库）
├── 📁 [科目名称]
│   ├── 📄 知识图谱
│   ├── 📄 学习计划
│   ├── 📁 第1章 - [模块名]
│   │   ├── 📄 第1节 - [知识点]
│   │   └── ...
│   ├── 📁 第2章 - [模块名]
│   ── 📁 复习资料 / 报表
── 📁 [另一科目]
│   └── ...
└── 📁 复盘报表
```

### 多维表格数据库

5 张数据表记录学习全过程，内置图表生成 Dashboard：

| 数据表 | 用途 |
|--------|------|
| 知识节点表 | 知识图谱结构 + 掌握度 + 状态 |
| 学习记录表 | 每次学习的时长、效率、自评 |
| 测验记录表 | 每次测验的成绩、掌握度变化 |
| 错题本表 | 错题管理 + 间隔复习计划 |
| 掌握度追踪表 | 掌握度时间序列（趋势图数据源）|

## 多科目并行

支持同时学习多个科目，每个科目独立管理：

```
我还想学 [另一个科目]
```

Agent 会为每个科目创建独立的知识树、学习计划、进度追踪，互不干扰。

## 定时任务

| 任务 | 默认时间 | 内容 |
|------|---------|------|
| 每日学习推送 | 工作日 9:00 | 推送新内容 |
| 每日复习提醒 | 工作日 20:00 | 提醒复习 |
| 周末学习推送 | 周六 9:00 | 推送新内容 |
| 每周测验 | 周五下午 | 阶段测验 |
| 每周复盘 | 周五傍晚 | 生成周报 |
| 每月复盘 | 每月 1 号 | 生成月报 |

随时调整：_"改成每天晚上 8 点推送"_、_"周末不推送"_、_"增加测验频率"_

## 考前冲刺模式

距离考试 ≤ 14 天时自动切换（有标准型）：暂停非核心内容，聚焦高频考点和易错点，每日推送量 1.5 倍，测验每日 1 次，每日 3 次推送，每次测验前生成"必复习清单"。

对于无标准型目标，最近里程碑到期 ≤ 7 天时也会进入温和冲刺模式：聚焦当前里程碑覆盖的知识点，推送量适度增加，增加验收模拟练习。

## 常见问题

**推送的内容太简单/太难？** 直接告诉 Agent，会调整后续内容的深度和长度。

**没时间学想暂停？** 说"暂停学习计划"，Agent 会暂停推送并顺延计划。

**想修改知识树？** 直接说"增加/删除 XXX"，Agent 会更新知识树并同步飞书目录。

**飞书文档创建失败？** 内容保存本地并通知你，授权后自动补推。

**能导出学习数据吗？** 飞书多维表格可直接导出 Excel/CSV。

**多人共用？** 每个用户通过飞书 open_id 自动识别，数据完全隔离。

## 技术信息

- **平台**：OpenClaw Agent
- **渠道**：飞书 WebSocket 长连接（流式输出已开启）
- **模型**：推荐使用推理模型（如 `bailian-thinking/qwen3.7-plus`），在 openclaw.json 中配置
- **工具**：飞书全套（文档、多维表格、知识库、消息、日历、任务等）
- **权限**：180+ scope，首次创建学习项目时一次性授权（详见 `workspace/templates/feishu-scopes.json`）
- **技能模块**：11 个 Skill
- **部署**：`bash setup.sh` 一键部署 / AI 工具部署见 `docs/DEPLOY-PROMPT.md`
